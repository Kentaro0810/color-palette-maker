# color-palette-maker
# カラーパレット作成ツール

ランダムにカラーパレットを生成できるWebアプリです。

## 機能

- 5色のカラーパレットをランダム生成
- HEXカラーコードを表示
- HEXカラーコードをクリックでコピー
- 色のロック機能
- CSS変数としてコピー
- お気に入りパレットの保存
- お気に入りパレットの読み込み
- お気に入りパレットの削除

## 使い方

1. 「ランダム生成」を押す
2. 気に入った色があれば「ロック」を押す
3. もう一度「ランダム生成」を押す
4. 必要ならHEXコードやCSS変数をコピーする
5. 気に入った配色は「お気に入り保存」を押す
